### IDE插件

#### 新增特性

    支持组织名语法高亮和补全；
    支持中心仓路径，开发者可以配置中心仓依赖并提供相关的语言服务。

### cjpm

#### 新增特性

- 适配组织名新语法
- 支持用户配置构建脚本产物路径
- cjpm run 支持 -- 标记入参
- 新增 bundle 命令，支持验证并打包本地项目
- 支持中心仓制品包上传（中心仓网址暂未开放，后续统一提供）
- 支持中心仓制品包下载（中心仓网址暂未开放，后续统一提供）
