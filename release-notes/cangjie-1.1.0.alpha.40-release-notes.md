## 编译器

### 新增特性

* Android 交叉编译及运行平台支持范围扩大至 API level 26 及以上，支持 `--target=aarch64-linux-android26` 编译选项（仅支持 Android 交叉编译的仓颉 SDK 可用）
* 提供 OHOS 版本仓颉 SDK，支持在鸿蒙 PC 运行仓颉编译器进行仓颉代码的编译

## 标准库

### 新增特性

- 新增可以通过反射获取包的组织名。
- 新增适配鸿蒙PC。
- 跨平台开发 FFI-ObjC 支持仓颉映射 ObjC 顶层函数，以及在仓颉侧调用。
- 跨平台开发 FFI-ObjC 支持仓颉侧使用 init* 构造对象。
- 跨平台开发 FFI-ObjC 支持仓颉侧使用 Option 映射 ObjC 指针类型。
- 跨平台开发 FFI-ObjC 支持映射 ObjC 的 C Struct 类型。
