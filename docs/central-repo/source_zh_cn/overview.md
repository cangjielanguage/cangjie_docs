# 概述

**仓颉中心仓 (Cangjie Central Repository)**（以下简称“中心仓”）是仓颉语言的三方库托管平台。

中心仓由以下三个部分组成：

- 中心仓官网：用于搜索、浏览制品信息，以及对用户和组织的管理操作；
- 中心仓客户端：用于制品的上传和使用的命令行工具，集成在仓颉项目管理工具 `cjpm` 中，请参阅[仓颉项目管理工具使用指南](https://gitcode.com/Cangjie/cangjie_docs/blob/dev/docs/tools/source_zh_cn/cmd-tools/cjpm_manual.md)以了解更多关于 `cjpm` 的信息；
- 中心仓仓库：用于存放三方库制品包及对应元数据。
