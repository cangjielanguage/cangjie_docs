# Overview

**Cangjie Central Repository** (hereinafter referred to as the "central repository") is the third-party library hosting platform for the Cangjie language.

The central repository consists of the following three components:

*   **Central Repository Website**: Used for searching and browsing artifact information, as well as managing users and organizations.
*   **Central Repository Client**: A command-line tool integrated into the Cangjie Project Manager `cjpm`, used for uploading and consuming artifacts. Please refer to the [Cangjie Project Manager User Guide](https://gitcode.com/Cangjie/cangjie_docs/blob/dev/docs/tools/source_en/cmd-tools/cjpm_manual.mdl) for more information about `cjpm`.
*   **Central Repository Storage**: Used for storing third-party library artifact packages and their corresponding metadata.
